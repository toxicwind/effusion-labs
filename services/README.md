# Effusion Labs Services

This directory (`services/`) consolidates the various micro-services and tools used in the Effusion Labs monorepo.

## Structure

- **`markdown-gateway/`**: A Python Flask service for converting URLs to Markdown using FlareSolverr.
- **`mcp-stack/`**: Shell-based CI/CD and Gateway infrastructure.
- **`openapi.yaml`**: Unified API definition for these services.

## Usage

Run services individually or use the root `npm run services:start` (coming soon).
